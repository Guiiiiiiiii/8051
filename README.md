## 開發環境
項目                |工具
:-------------------|:----------------
程式語言            |C
單晶片              |AT89S52
開發軟體            |Keil C
PCB繪圖軟體         |Altium Designer

## 開發項目
* 項目1 4x4鍵盤控制三色燈
* 項目2 上傳溫濕度到雲端
* 項目3 PWM調光
* 項目4 溫度警報器
* 項目5 讀取溫濕度值
* 項目6 3路PWM輸出
